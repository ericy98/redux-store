# React Portfolio
https://floating-cove-02485.herokuapp.com/

## Description:
A refactored e-commerce website to use Redux for state management. 

## Installation:
Download node.js

To clone all the dependencies:
 > npm install

Run seed data before starting server:
 > npm run seed

Start the server:
  > npm start
